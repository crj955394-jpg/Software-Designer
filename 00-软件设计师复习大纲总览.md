# 软件设计师考试复习大纲总览

## 考试说明
- **考试级别**: 中级
- **考试形式**: 笔试（上午+下午）
- **上午考试**: 基础知识（75道选择题，每题1分，共75分）
- **下午考试**: 应用技术（5-6道大题，共75分）
- **合格标准**: 上午和下午均需达到45分（含）以上

---

## 一、计算机系统知识（约15%）

### 1.1 计算机硬件基础
- 1.1.1 数据的表示（原码、反码、补码、移码、浮点数、定点数）
- 1.1.2 校验码（奇偶校验、海明码、CRC循环冗余校验）
- 1.1.3 计算机组成（运算器、控制器、存储器、输入输出设备）
- 1.1.4 流水线技术（吞吐率、加速比、效率）
- 1.1.5 存储系统（Cache、主存、辅存，层次结构）
- 1.1.6 总线系统
- 1.1.7 可靠性与系统性能评价（MTTF、MTTR、MTBF）

### 1.2 计算机体系结构
- 1.2.1 Flynn分类法（SISD、SIMD、MISD、MIMD）
- 1.2.2 CISC与RISC
- 1.2.3 流水线技术
- 1.2.4 存储管理

### 2.1 编译与解释

- 2.1.2 编译器与解释器的区别

- 2.2.2 静态作用域与动态作用域

## 三、数据结构与算法（约15%）

### 3.1 线性结构
- 3.1.1 数组
- 3.1.2 线性表（顺序存储、链式存储）
- 3.1.3 栈和队列
- 3.1.4 串

### 3.2 树和图
- 3.2.1 二叉树（满二叉树、完全二叉树、二叉排序树、平衡二叉树）
- 3.2.2 树的遍历（前序、中序、后序、层次遍历）
- 3.2.3 哈夫曼树与哈夫曼编码
- 3.2.4 图的存储（邻接矩阵、邻接表）
- 3.2.5 图的遍历（深度优先、广度优先）
- 3.2.6 最小生成树（Prim算法、Kruskal算法）
- 3.2.7 最短路径（Dijkstra算法、Floyd算法）
- 3.2.8 拓扑排序与关键路径

### 3.3 查找与排序
- 3.3.1 查找算法（顺序查找、二分查找、哈希查找）
- 3.3.2 排序算法（冒泡、选择、插入、快速、归并、堆排序）
- 3.3.3 算法时间复杂度与空间复杂度分析

### 3.4 算法设计方法
- 3.4.1 分治法
- 3.4.2 动态规划
- 3.4.3 贪心算法
- 3.4.4 回溯法
- 3.4.5 分支限界法

## 复习要点总结

### 重点章节（需深入掌握）
1. **数据结构与算法** - 考试重点，必考内容多
2. **软件工程** - 下午题主要考查对象
3. **数据库系统** - 必考内容，分值较高
4. **计算机网络** - 常考知识点
5. **面向对象技术** - UML与设计模式是重点

### 常考题型
1. 数值计算题（数制转换、校验码、浮点数）
2. 流水线计算题（吞吐率、加速比）
3. 可靠性计算题（MTTF、MTBF）
4. 数据结构题（树、图的遍历）
5. 排序算法题
6. 操作系统调度题（进程调度、页面置换、磁盘调度）
7. 数据库规范化题
8. SQL语句题
9. 网络地址计算题
10. UML图分析题
11. 设计模式识别题

### 学习建议
1. 先通读教材，建立知识框架
2. 重点章节反复学习，做到理解透彻
3. 多做历年真题，熟悉考试题型
4. 总结常见考点和解题方法
5. 下午题需要多练习，注重规范性和完整性
6. 合理安排时间，上午题要控制答题速度
