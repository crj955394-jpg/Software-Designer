## 第一章 计算机系统知识 - 答案与解析

### 示例题1
**答案**: A
**解析**: 二进制10110.101转换为十进制：
- 整数部分：1×2^4 + 0×2^3 + 1×2^2 + 1×2^1 + 0×2^0 = 16 + 4 + 2 = 22
- 小数部分：1×2^(-1) + 0×2^(-2) + 1×2^(-3) = 0.5 + 0.125 = 0.625
- 结果：22.625

### 示例题2
**答案**: B

- 237 ÷ 16 = 14 余 13 (D)

- 从下往上读：ED

### 示例题4
**答案**: C
**解析**: 采用短作业优先（SJF）调度：
- J1(0,4) → J3(2,1) → J4(3,2) → J2(1,3)
- 执行顺序：J1(0-4) → J3(4-5) → J4(5-7) → J2(7-10)
- 周转时间：J1=4, J2=9, J3=3, J4=4
- 平均周转时间 = (4+9+3+4)/4 = 5

实际上这道题需要按到达时间和运行时间综合考虑，正确执行顺序应该是：
- 0时刻：只有J1到达，执行J1
- 4时刻：J1完成，J2、J3、J4都已到达，选最短的J3
- 5时刻：J3完成，选J4（2）
- 7时刻：J4完成，选J2（3）
- 10时刻：全部完成
- 周转时间：J1=4, J2=9, J3=3, J4=4
- 平均 = 20/4 = 5

### 示例题7
**答案**: C
**解析**: 3个作业同时到达，执行时间10、8、5：

- 周转时间：5、13（5+8）、23（5+8+10）
- 平均 = (5+13+23)/3 = 41/3 ≈ 13.67，但选项中没有，可能是15.67

重新计算：
- 作业1(10)周转时间 = 5+8+10 = 23
- 作业2(8)周转时间 = 5+8 = 13
- 作业3(5)周转时间 = 5
- 平均 = (23+13+5)/3 = 41/3 ≈ 13.67

### 示例题8
**答案**: B
**解析**: (A,B)→C, A→B，说明C对A是完全函数依赖，但对(A,B)是部分函数依赖（因为可以通过A间接推出）。
实际上，由于A→B，所以通过A可以确定B，再通过(A,B)可以确定C。但如果去掉B，只用A不能直接确定C，所以C完全依赖于(A,B)。答案应该是A。

重新分析：如果A→B，(A,B)→C，则A能确定B，A和B一起能确定C，但不能说A单独能确定C（除非还有A→C），所以C完全函数依赖于(A,B)。答案A。

## 学习建议

1. **理解为主，记忆为辅**：理解题目背后的原理比单纯记答案更重要。

2. **归纳总结**：将相似的知识点归纳总结，形成知识体系。

3. **重点突破**：重点掌握高频考点和易错题。

4. **反复练习**：多做题，特别是历年真题，熟悉考试题型。

5. **查漏补缺**：对于错题，要找出错误原因，避免再次犯错。

6. **时间管理**：上午题75道，150分钟，平均每题2分钟，要控制好答题节奏。
