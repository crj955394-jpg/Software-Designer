## 4.1 进程管理

### 4.1.1 进程的基本概念

**知识点**:
- 进程是程序的一次执行过程
- 进程的三态模型：运行态、就绪态、阻塞态（等待态）
- 进程的五态模型：新建态、就绪态、运行态、阻塞态、终止态
- 进程控制块（PCB）：存储进程信息

### 4.1.2 进程调度算法

**知识点**:
1. **先来先服务（FCFS）**：
   - 按到达顺序调度
   - 简单，但平均等待时间长
   - 非抢占式

2. **短作业优先（SJF）**：
   - 选择执行时间最短的作业
   - 平均等待时间最短
   - 可能导致长作业饥饿

3. **优先级调度**：
   - 按优先级调度
   - 可能导致低优先级进程饥饿

4. **时间片轮转（RR）**：
   - 每个进程分配一个时间片
   - 时间片用完则切换
   - 抢占式，公平

5. **多级反馈队列**：
   - 结合多种调度算法
   - 动态调整优先级

### 4.1.3 进程同步与互斥

**知识点**:
- **临界资源**：一次只允许一个进程访问的资源
- **临界区**：访问临界资源的代码段
- **互斥**：多个进程不能同时进入临界区
- **同步**：进程按一定顺序执行
- **信号量（Semaphore）**：
  - P操作（wait）：S = S - 1，若S < 0则阻塞
  - V操作（signal）：S = S + 1，若S ≤ 0则唤醒一个进程
- **PV操作的应用**：
  - 互斥：初值为1
  - 同步：初值为0或资源数

### 4.1.4 死锁

**知识点**:
- **死锁产生的四个必要条件**：
  1. 互斥：资源不能共享
  2. 占有并等待：持有资源并等待其他资源
  3. 不可剥夺：资源不能强制剥夺
  4. 循环等待：存在循环等待链

- **死锁处理**：
  1. 预防：破坏四个条件之一
  2. 避免：银行家算法
  3. 检测：资源分配图
  4. 解除：剥夺资源、撤销进程

**示例题11**:

## 4.2 存储管理

### 4.2.1 分区存储管理

**知识点**:
- **固定分区**：内存划分为若干固定大小的分区
  - 简单，但内存利用率低
  - 产生内部碎片

- **可变分区（动态分区）**：
  - 按需划分分区
  - 产生外部碎片
  - 分配算法：
    - 首次适应（FF）：找到第一个满足的分区
    - 最佳适应（BF）：找到最小的满足的分区
    - 最坏适应（WF）：找到最大的分区

**示例题15**:

### 4.2.2 页式存储管理

**知识点**:
- 逻辑地址空间和物理地址空间都划分为固定大小的页（页框）
- 页面大小通常为2的幂次，如4KB
- 页表：记录页号到物理块号的映射
- 地址转换：逻辑地址 = 页号 + 页内偏移
- 优点：无外部碎片，内存利用率高
- 缺点：有内部碎片

### 4.2.3 段式存储管理

**知识点**:
- 按逻辑段划分，每段大小可以不同
- 段表：记录段号到物理地址的映射
- 地址转换：逻辑地址 = 段号 + 段内偏移
- 优点：便于共享和保护
- 缺点：产生外部碎片

**示例题21**:

### 4.2.4 段页式存储管理

**知识点**:
- 结合段式和页式的优点
- 先分段，再分页
- 地址转换：段号 → 段表 → 页号 → 页表 → 物理地址

### 4.2.5 虚拟存储

**知识点**:
- 将部分程序和数据装入内存，其余放在外存
- 请求分页：按需调页
- 页面置换算法：
  1. **最佳置换（OPT）**：置换未来最长时间不用的页（理论最优）
  2. **先进先出（FIFO）**：置换最先进入的页
  3. **最近最少使用（LRU）**：置换最长时间未使用的页
  4. **时钟算法（Clock）**：循环扫描，使用访问位

- 缺页率 = 缺页次数 / 总访问次数

## 4.3 设备管理

### 4.3.1 I/O控制方式

**知识点**:
1. **程序直接控制（查询方式）**：
   - CPU不断查询设备状态
   - CPU利用率低

2. **中断方式**：
   - 设备准备好后发出中断
   - CPU可以处理其他任务

3. **DMA方式**：
   - 数据直接在内存和设备间传输
   - CPU只需处理开始和结束

4. **通道方式**：
   - 通道是独立的处理器
   - 专门负责I/O操作

**示例题27**:
在I/O控制方式中，( )方式下CPU的效率最高。

### 4.3.2 缓冲技术

**知识点**:
- **单缓冲**：一个缓冲区
- **双缓冲**：两个缓冲区，一个输入一个输出
- **循环缓冲**：多个缓冲区组成循环队列
- **缓冲池**：系统管理的缓冲区集合

**示例题29**:

### 4.3.3 SPOOLing技术

**知识点**:
- 假脱机技术
- 将独占设备改造为共享设备
- 典型应用：打印机管理
- 由输入井、输出井、输入进程、输出进程组成

## 4.4 文件管理

### 4.4.1 文件的逻辑结构

**知识点**:
1. **顺序文件**：记录按顺序存储
2. **索引文件**：建立索引表
3. **索引顺序文件**：分组+索引
4. **直接文件（散列文件）**：通过哈希函数确定位置

### 4.4.2 文件的物理结构

**知识点**:
1. **连续分配**：
   - 文件占用连续的磁盘块
   - 顺序访问快
   - 会产生碎片

2. **链接分配**：
   - 每个磁盘块有指向下一块的指针
   - 不产生碎片
   - 随机访问慢

3. **索引分配**：
   - 用索引块存储所有数据块的地址
   - 支持随机访问
   - 需要额外空间存储索引

**示例题32**:

1. 进程的状态转换

2. PV操作应用题
3. 死锁判断题
4. 页面置换算法模拟题
5. 地址转换计算题
6. 磁盘调度计算题

### 重要公式
1. **周转时间** = 完成时间 - 到达时间
2. **等待时间** = 周转时间 - 运行时间
3. **带权周转时间** = 周转时间 / 运行时间
4. **平均周转时间** = 所有作业周转时间之和 / 作业数
5. **页号** = 逻辑地址 / 页面大小
6. **页内偏移** = 逻辑地址 % 页面大小
7. **缺页率** = 缺页次数 / 总访问次数
8. **物理地址** = 块号 × 页面大小 + 页内偏移
